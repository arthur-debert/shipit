`shipit fleet sweep` runs applicable tool verbs against each portfolio repo in
fresh Trees.

Applicability comes from each target repo's own declarations: lint always
applies, test and build apply where toolchains declare legs, e2e applies where
artifacts declare harnesses, and changelog check applies where changelog
fragments exist. Each executed cell carries Work Env evidence (`fleet-sweep.cell`,
WorkingDir/Tree coordinates, checkout strategy, routing, repo, and tool) so a
reader can tell whether it ran through pixi or ambient tools without recording a
full environment snapshot or fabricated pixi run id. The output is an evidence
matrix with raw output for red cells.
